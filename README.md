# clawgame

2D Android Sandbox Game Prototype.

## Concept
- 2D Top-down (Pokemon-style 2D movement/terrain).
- Sandbox world with limited boundaries.
- Randomly generated terrain.
- Hotbar (5 slots) + Inventory/Crafting grid.
- Controls: Movement (U/D/L/R), Hit, Interact.

## Prototype Goals
- Procedural generation (Trees, Bushes).
- Breakable Trees.
- Decorative Bushes.
- Craftable Wooden Tools (Sword, Axe, Pickaxe).
